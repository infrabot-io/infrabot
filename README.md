## :newspaper: Description

<img src="assets/infrabot.png" height="20" width="20"/> infrabot - is an on-premise service which allows you to create your own commandlets to control your infrastructure using a Telegram Bot. Built-in tools will allow to create, configure and manage plugins and application configuration.

## :vhs: Plugin support

To add functionality to infrabot and extend its capabilities, plugin system was implemented. Each plugin determines which command infrabot can execute. One plugin - means one commandlet. See [examples](https://infrabot-io.github.io/documentation/examplescripts.html) to get to know how to create plugins.
